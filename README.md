# BookStore
网上书店：JAVA企业级开发课程的课程设计，由自己独立完成。
开发工具：eclipse/ Tomcat
前端部分：Bootstrap/ JQuery/ Ajax/ CSS/ JSP
服务器：Servlet
数据库：MYSQL/ JDBC
此项目排版模仿读读狗网站，实现了用户购买与下载图书的功能。

数据库可通过eneity包中的类自行完成。
